ExPolygons

Del.» TExPolygons = array of ExPolygon;

C++» typedef std::vector< ExPolygon > ExPolygons;

C#» using ExPolygons = List< ExPolygon >;


ExPolygons encapsulates one or a number of ExPolygon structures.

The solution parameter in Clipper's overloaded Execute method can return either a Polygons structure or an ExPolygons structure.
